# RYM ratings to /10 (Chrome Extension)

A Chrome extension that converts RateYourMusic ratings from /5 to /10 by doubling the displayed value.

## Dev install
1. Open Chrome → `chrome://extensions`
2. Enable **Developer mode**
3. Click **Load unpacked**
4. Select the `src/` folder
5. Visit https://rateyourmusic.com and open DevTools → Console to confirm the script loaded.
